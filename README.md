# boa.vim
Colorscheme for vim and neovim.
It works only in terminals with 24-bit colour support.

-----------

### Python

![Python](http://i.imgur.com/VZlAOSe.png)

Instruction
-----------

Add this lines to your .vimrc or init.vim(for neovim)
```
set termguicolors
set background=dark
colorscheme boa
```

Configured for
--------------

- CSS(CSS3, Less, Sass)
- HTML/HTML5
- JavaScript
- Markdown
- Python/Django
- Shell
